To run this application:

- Copy the postgresql-8.4-701.jdbc3.jar file into <TOMCAT_HOME>/lib of your
  Tomcat installation.

- Copy the webapp/ directory to <TOMCAT_HOME>/webapps/data-source-realm of your
  Tomcat installation.

- Run the SQL script ddl.sql in pgAdmin to create the tables / data for user
  names and roles.
